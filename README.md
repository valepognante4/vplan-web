VPlan 🚀
VPlan es una aplicación web diseñada para la gestión de tareas personales, enfocada en la productividad y la claridad visual.

📋 Descripción
Plataforma que permite a los usuarios organizar, priorizar y alcanzar sus metas diarias mediante un panel intuitivo y moderno.

🛠 Tecnologías Utilizadas
Frontend: HTML5, CSS3, JavaScript.
Despliegue: Vercel.

🌐 Acceso al Proyecto
Puedes ver la aplicación en funcionamiento aquí: https://vplan-web.vercel.app/
